Projects

Stock Technical Analysis using Python

Self-directed project using 20, 50, and 200 day SMA, 20 and 50 day EMA, Bollinger Bands, RSI, Stochastic Oscillator, and Fibonacci Retracement Levels to generate both short and long-term stock analysis.

Future plans include developing a front-end with more features.

Portfolio Optimization using Hierarchical Risk Parity in Python

Self-directed project. Built a portfolio optimizer using HRP for effective capital allocation and risk reduction. Utilized `yfinance` for dynamic stock data retrieval and `pypfopt` for optimization.

A Lucid Dream (2D Computer Game)

Early Detection of Breast Cancer using AI
