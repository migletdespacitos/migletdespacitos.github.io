About Me

Hello! I'm Miguel, a Year 3 undergraduate at the National University of Singapore, majoring in Data Science & Analytics. I am passionate about Mathematics and Statistics and have a deep interest in machine learning and big data analytics. I thrive on uncovering insights from complex datasets and am enthusiastic about applying my skills to real-world challenges, contributing to innovative solutions and advancements in the field of data science.

Projects

Stock Technical Analysis using Python

Self-directed project using 20, 50, and 200 day SMA, 20 and 50 day EMA, Bollinger Bands, RSI, Stochastic Oscillator, and Fibonacci Retracement Levels to generate both short and long-term stock analysis.

Future plans include developing a front-end with more features.

Portfolio Optimization using Hierarchical Risk Parity in Python

Self-directed project. Built a portfolio optimizer using HRP for effective capital allocation and risk reduction. Utilized `yfinance` for dynamic stock data retrieval and `pypfopt` for optimization.

Book Recommendation App

Developed backend for a book recommendation system using Python, Pandas, and Kaggle as part of NUS Hackers Hack&Roll 2024.

A Lucid Dream (2D Computer Game)

Developed a 2D roguelike computer game using Unity, C#, and Firebase as part of CP2106 Independent Software Development.

Early Detection of Breast Cancer using Machine Learning

Worked with a team to train Machine Learning models for early breast cancer detection using K-nearest Neighbours, Logistic Regression, and Neural Networks as part of IT1244: Artificial Intelligence: Technology and Impact. Tools used include Python, NumPy, sklearn, and Keras.
